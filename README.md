# Calculadora de Plano de Saúde

Projeto simples em Python que calcula o valor de um plano de saúde com base na idade do usuário.

## Funcionalidades

* Entrada de valor base
* Cálculo por faixa etária
* Exibição do valor final

## Tecnologias

* Python

## Objetivo

Praticar lógica de programação e estruturas condicionais.
